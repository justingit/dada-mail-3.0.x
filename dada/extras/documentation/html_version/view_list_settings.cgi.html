View List Settings

This plugin allows you to view what ALL your list settings are set to

$ADMIN_MENU ENTRY

         {-Title      => 'View List Settings', 
          -Title_URL  => "plugins/view_list_settings.cgi",
          -Function   => 'view_list_settings',
          -Activated  => 1, 
          },
  
=cut
