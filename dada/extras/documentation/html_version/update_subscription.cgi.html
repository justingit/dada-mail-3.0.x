NAME update_subscriptions.cgi

DESCRIPTION

The script allows a subscriber to change the address they are subscribed to another address for all lists at once.

INSTALLATION

upload this script in plaintext, or ASCII, mode into the extensions directory inside the dada directory, something like:

 /home/account/cgi-bin/dada/extensions

If this directory doesn't exist, make it.

chmod 755 this script.

You are good to go.

View this script in a web browser, if everything is installed correctly, you should see a subscription update form.
