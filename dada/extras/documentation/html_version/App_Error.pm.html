NAME

DADA::App::Error

SYNOPSIS

        use DADA::App::Error

This module basically has error messages in HTML and spits 'em back at ya.

SUBROUTINES

cgi_user_error

 print cgi_user_error(-List  => 'my_list', 
                      -Error => 'some_error', 
                      -Email => 'some@email.com');

Gives back an HTML friendly error message.
